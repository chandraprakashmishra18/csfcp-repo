# Project Overview:
I made this project as it helped me in improving my skills in HTML and CSS. This project helped me learn advanced CSS Concepts. 
It showcases my personal information, skills, projects, and screenshots in a clean and organized layout.

The goal of this project is to design a simple, responsive, and visually appealing webpage that represents my identity as a student and beginner developer. The website contains multiple sections including a header, about section, projects, and contact information.

The project was developed using Visual Studio Code (VS Code) and version-controlled using Git. The complete source code is hosted on GitHub. This project helped me understand the practical usage of development tools such as Git commands, repository management, commits, folder structure, and documentation using Markdown.
# Key Features

HTML-based structure for clean organization

CSS styling for layout, colors, spacing, and visuals

Images section containing screenshots and profile images

Responsive and simple design suitable for beginners

Well-organized GitHub repository with proper folders and documentation

README file providing usage instructions and screenshots
# Tools & Technologies Used

HTML5 — to structure the webpage

CSS3 — to style and format the content

Visual Studio Code — development environment

Git — version control for commits and tracking changes

GitHub — for hosting the project repository and documentation
# Purpose of This Project

The purpose of this mini-project is to:

Practice coding using HTML and CSS

Learn and apply Git commands such as git init, git add, git commit, and git push

Understand how to manage a project on GitHub

Organize files properly using folders

Write documentation using Markdown

Build confidence in working with real-world development tools
