# Development Process

This portfolio website was developed using a step-by-step approach:

Planning the layout

Decided the sections needed (About, Skills, Projects, Contact).

Sketched a basic structure for the webpage.

Creating the HTML structure

Built the main layout using semantic HTML tags.

Added content such as text, images, and links.

Styling with CSS

Created a clean and simple design using CSS.

Added colors, fonts, margins, padding, and layout adjustments.

Designed the page to be responsive for different screen sizes.

Organizing assets

Moved images into a dedicated images/ folder.

Created a separate style.css file for clean code organization.

Version control using Git & GitHub

Initialized a Git repository in VS Code.

Committed files and pushed the project to a GitHub repository.

Resolved merge conflicts and updated files as needed.

Testing the website

Checked the website in a browser.

Ensured all links, images, and styles worked correctly.

Made corrections and improvements based on feedback.
